# arduino_small_projects
아두이노를 활용한 작은 프로젝트들

## Random Button for Game
버튼을 누르면 랜덤으로 3개중 하나의 LED가 선택되는 시스템이며, 가위바위보 혹은 랜덤으로 뭔가를 해야할 때 활용할 수 있다.
